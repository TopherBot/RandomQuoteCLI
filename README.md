# RandomQuoteCLI

A minimal command‑line utility that prints a random inspirational quote.

## How to use
```bash
$ python random_quote.py
"The only limit to our realization of tomorrow is our doubts of today." – Franklin D. Roosevelt
```

## Why?
- Instant positivity in 1 KB of source.
- No external packages; works on any Python 3.8+ installation.
- Great starter for learning file I/O and random selection.
